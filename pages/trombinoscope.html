Trombinoscope

Informations du projet

  • Catégorie: Application Web & Interface d'Administration
  • Client: CinetPay
  • Date de publication: 05 Février 2021

Description

Il s'agit d'un système permettant de recceuillir l'ensemble des informations de tous les employés dans l'entreprise.

Le système est capable d'identifier les employé selon certains critères tels que :

  • Le Pays
  • La Direction
  • La Fonction
  • Etc ...
