Description

Il s'agit ici de mettre en place un système permettant aux assurées d'acheter votre bon d'achat, imprimez votre reçu et votre bon, puis rendez vous dans un centre SOCOCE pour utiliser votre bon.

Il existe trois (03) types d’organismes d’assurances, régies par trois codes de droit distincts :

  • Les entreprises d’assurances
  • Les institutions de prévoyance
  • Les mutuelles
